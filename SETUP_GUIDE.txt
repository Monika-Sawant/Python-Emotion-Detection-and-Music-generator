# EMOTION DETECTION SYSTEM - SETUP GUIDE

## Installation Instructions

### Step 1: Prerequisites
Ensure you have Python 3.8 or higher installed.
- Download Python from https://www.python.org/downloads/
- Verify installation: `python --version`

### Step 2: Install Dependencies
Run the following command to install all required libraries:

```bash
pip install -r requirements.txt
```

### Step 3: Complete Installation (One-time setup)
The application will automatically download the required machine learning models on first run.

## System Requirements

### Hardware
- Processor: Intel i3 or equivalent (or higher)
- RAM: 4GB minimum (8GB recommended)
- Storage: 2GB free space for models
- Webcam: Required for facial recognition feature
- Microphone: Required for voice input feature

### Software
- Windows 7+, macOS 10.12+, or Linux
- Python 3.8+
- pip (Python package manager)

## Running the Application

### Windows
```bash
python emotion_detector.py
```

### macOS/Linux
```bash
python3 emotion_detector.py
```

## Troubleshooting

### Camera Not Working
- Check if camera is connected and accessible
- Allow application permission to access camera
- Try restarting the application

### Microphone Issues
- Verify microphone is connected and set as default input device
- Test microphone in system settings
- Ensure microphone has proper permissions

### Model Download Issues
- Check internet connection
- Models will be downloaded automatically (requires ~500MB free space)
- First run may take 2-3 minutes to download models

### ImportError
- Ensure all requirements are installed: `pip install -r requirements.txt`
- Use `pip install --upgrade pip` if having issues

## Features Overview

### 1. Facial Recognition
- Detects face and analyzes emotion in real-time
- Generates instrumental music based on detected emotion
- Live camera feed display

### 2. Text Analysis
- Analyze emotion from typed text
- Supports sentences and paragraphs
- Instant emotion classification

### 3. Voice Input
- Record voice input (up to 15 seconds)
- Automatic speech-to-text conversion
- Emotion detection from spoken words

### 4. Music Generation
- Generates unique instrumental music (16 seconds)
- Different compositions for each emotion:
  - Happy: Major scale, uplifting tempo
  - Sad: Minor scale, slower tempo
  - Angry: Dissonant harmonics, fast tempo
  - Neutral: Balanced, moderate tempo
- Real-time playback without file saving

## Performance Notes

- First run: 2-3 minutes (model download and setup)
- Subsequent runs: Instant startup
- Facial recognition: ~30 FPS on modern hardware
- Text analysis: <1 second per analysis
- Voice processing: Real-time
- Music generation: ~5-10 seconds

## Data Privacy

- No data is stored or transmitted
- All processing is done locally
- Camera feed is not recorded or saved
- Voice is not stored (only used for text conversion)
- Models are downloaded from Hugging Face (open-source)

## Model Information

The system uses:
- DistilBERT for sentiment analysis (569 MB)
- OpenCV Haar Cascades for face detection (built-in)
- Scipy/NumPy for audio synthesis

All models are completely free and open-source.
