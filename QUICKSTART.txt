❓ COMMON ISSUES & QUICK FIXES

Issue: "ModuleNotFoundError" or "No module named..."
→ Solution: Run "pip install -r requirements.txt" again

Issue: Camera not showing
→ Solution: 
  1. Allow app permission for camera (check system settings)
  2. Restart application
  3. Use different USB port if external camera

Issue: Microphone not recording
→ Solution:
  1. Check microphone is set as default input
  2. Test mic with another app first
  3. Ensure no other app is using mic

Issue: Very slow on first run
→ Solution: Normal! ML models (569MB) downloading. Wait 2-3 minutes.
           Next run will be instant.

Issue: Sound quality is poor
→ Solution: Close other audio apps, check system volume

🎵 MUSIC GENERATION

The system generates 16 seconds of instrumental music based on emotion:

NEUTRAL: Balanced, moderate, steady sound

✓ 44.1 kHz quality stereo audio
